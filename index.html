Fuhrat Robot
Trip Personal Assistant Robot: Atlas

Atlas is a conversational travel assistant designed to help users plan their journeys and explore destinations.

It utilizes the Furhat API to create interactive and engaging conversations with users. The assistant provides information about travel destinations, tips, and allows users to express emotions through Conversation.

GIS, NoSQL
Spatial Data Analysis for Urban Transportation

This project aimed to analyze a bus dataset for urban transportation insights by utilizing various database and GIS techniques.

NoSQL databases, including Key-Value Stores and Document Databases, along with data wrangling and GIS methods, were used to process and visualize the data.

Github Link

RNN, GRU
English-French Translator

The English-French Translator project uses a Recurrent Neural Network (RNN) with Gated Recurrent Units (GRU) for sequence-to-sequence translation of English sentences into French.

It involves data preprocessing, tokenization, model training, and the deployment of an intuitive user interface for real-time translations using Gradio.

Github Link

SVC, Flask
Medicine Recommendation System

System that predicts diseases based on symptoms provided by the user.

It utilizes a dictionary for symptom-to-disease mapping and a machine learning model (SVC) to predict the disease, followed by providing detailed information such as disease description, precautions, medications, diet plan, and recommended workouts.

Github Link
